Introducción

CarHub es una plataforma web diseñada para facilitar la compra y venta de vehículos nuevos y usados. Su enfoque en la transparencia y confianza la hace destacar en el mercado.

Objetivos

  • Facilitar el proceso de compraventa de vehículos.
  • Garantizar un entorno seguro mediante la verificación de identidades y vehículos.
  • Ampliar el alcance de concesionarios y empresas.
  • Garantizar transparencia en las transacciones.
  • Incorporar servicios complementarios como financiamiento y seguros.

Tecnologías

  • HTML: Estructura del contenido.
  • CSS: Diseño y presentación responsiva.
  • JavaScript: Interactividad y funciones dinámicas.
  • MySQL: Gestión de base de datos.
  • Futuras mejoras: React.js, Node.js o PHP.

Diagramas

Se incluyen diagramas UML de Casos de Uso y Clases para visualizar el sistema.

Esquema de Base de Datos

La base de datos de CarHub se organiza en cinco tablas principales: Usuarios, Vehículos, Historial, Financiación y Compra/Venta.

Recursos

El proyecto será desarrollado por estudiantes de DAW, utilizando Visual Studio Code y tecnologías front-end como HTML, CSS y JavaScript.
